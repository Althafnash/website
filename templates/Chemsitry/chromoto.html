Chromotography Explained

You can use paper chromotography to identifiy the particular dyes in a mixture

We can see that all the compounds in the mixture coresponding to the to the dyes has traveeled using this we can say that they are the same compounds

But if you look at dye 2 the compounds is not as the same distance as the compiounds in the mixture , we can say that the compound is a different compound

Each time the experiment is repeated the dyes will move differenently

we cannot work with the distance travelled we have to come up with a ratio

This ratio is known as the retradation factor

R1 = retradation factor

R1 = distance moved by a spot (From the pencil line ) / distance travelled by the solvent

Since the spots have the same R1 value as dye 1,2,3 we can say the three componets are dyes

R1 value must be between 0 and 1
