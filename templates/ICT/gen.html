Generation of computers

The computer was born in the attempt no to make an adding machine in order to add number a device called the abacus was invented

In 1642 blaise pascal invented a machine called the adding machine this was the world's first ever mathematical machine

In 1674 gottfried wilhelm von leibntiz improved the adding machine which could do multiplication and division

A french scientist named joseph jacquard invented a mechanical loom using the punch card system

Charles babbage stated to make his analytical engine using the punch card system concept This machine was based on the concept of input process store and output since this concept helps the development of the computer he was known as the father of computer

Madam ada agusta lovelace is considered to be the world's first programmer since she tried to write programs for the analytical engine
